<h1>TouhouRPG</h1>

TouhouRPG is a web-based online RPG coded in JavaScript, PHP, HTML and CSS using the framework Laravel.

<ul>
    <li>Create an account and edit your profile, chat with your friends, or show off your achievements!</li>
